# ImageOptimize Pro

**Free, privacy-first image compressor & converter running 100% in your browser**

Compress JPEG/PNG/WebP files, resize images, convert to WebP/JPEG – no server uploads, no data leaves your device.

Live demo: https://your-username.github.io/image-optimize-pro/

## Features
- Quality slider (10–95%)
- Optional max-width resize (preserves aspect ratio)
- WebP or JPEG output
- Side-by-side original vs optimized preview
- File size savings percentage shown
- Dark modern UI with drag & drop support

## Tech
- Pure HTML + CSS + Vanilla JavaScript
- Canvas API for processing
- No external libraries beyond Font Awesome CDN

## How to Run Locally
1. Download or clone this repo
2. Open `index.html` in any modern browser
